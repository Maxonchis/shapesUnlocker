[RU]
Для работы скрипта запускаем файл "runMe".
Далее указывай полный путь к папке. Например E:\Giants Engine\YourFolder.
Если скрипт находится рядом с вашей папкой, достаточно нажать Enter и он запустится из текущей папки.

[EN]
To run the script, run the "runMe" file.
Next, specify the full path to the folder. For example E:\Giants Engine\YourFolder.
If the script is next to your folder, just press Enter and it will run from the current folder.
